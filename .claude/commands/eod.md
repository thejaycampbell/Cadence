Run my end-of-day review.

## How to run this

1. Read:
   - `knowledge/00_user-brain/living-brain.md` — today's goals and decision rules
   - `memory/MEMORY.md` — any active project context
   - `inbox/` — anything unprocessed

2. Ask the user:
   - "What did you get done today? (Brief — bullets are fine)"
   - "What's still outstanding that didn't get done?"
   - "Anything that happened today worth capturing — a decision made, something you learned, a claim you want to keep?"

3. Process their answers:

   **Review today:**
   - What got done vs. what was planned
   - What got stuck and why
   - What to carry forward vs. kill

   **Tomorrow:**
   - Top 3 priorities for tomorrow based on outstanding items + goals

   **Knowledge capture:**
   - If they mentioned a decision or insight worth keeping: ask "Should I add that to knowledge/?" and if yes, add it to the appropriate file

   **Memory update:**
   - If anything new emerged (a person, a project update, a preference correction): update `memory/MEMORY.md`

4. Return:
